# 🌿 GardenGuru AI Bot

**GardenGuru** — це інтелектуальний помічник садівника, побудований на базі **LangGraph** та **OpenAI gpt-5-mini**. Проєкт використовує агентну архітектуру ReAct (Reason + Act), що дозволяє моделі самостійно приймати рішення про використання інструментів для вирішення завдань користувача.

---

## 🚀 Основні можливості

- 🌱 **Персоналізований догляд**: Надає інструкції щодо поливу, світла та ґрунту для конкретних рослин.
- 📅 **Інтерактивний календар**: Підказує, що і коли садити (на розсаду чи у відкритий ґрунт) залежно від місяця.
- 🔍 **Діагностика шкідників**: Аналізує симптоми (наприклад, "жовті плями на листі") та пропонує методи лікування.
- 📊 **Розумний калькулятор**: Розраховує точну вагу добрив у грамах залежно від площі ділянки та типу культури.
- 🧠 **Контекстна пам'ять**: Бот підтримує історію діалогу в межах сесії, дозволяючи ставити уточнюючі запитання.
- 🎨 **Кольорове логування**: Професійне відстеження роботи системи в терміналі.

---

## 📂 Структура проєкту

Проєкт організований за модульним принципом:

- `src/main.py` — Точка входу.
- `src/init.py` — Налаштування LangGraph, моделі та MemorySaver.
- `src/services.py` — Прошарок між інтерфейсом та AI-агентом.
- `src/tools.py` — Опис функцій-інструментів, якими користується бот.
- `src/schemas.py` — Pydantic-моделі для суворої валідації вхідних даних.
- `src/db.py` — Локальна база даних (плани, календарі, шкідники).
- `src/utils.py` — Системні налаштування та промпти.
- `src/logger_config.py` — Конфігурація кольорового виводу в консоль.

## 🛠 Технологічний стек

- **Python 3.14+**
- **LangGraph** (від LangChain) — для керування станами та циклами агента.
- **OpenAI API** — ядро штучного інтелекту.
- **Pydantic** — для валідації аргументів інструментів.
- **Colorlog** — для візуально зрозумілої розробки та дебагу.

## 💻 Як запустити (Terminal Mode)

1.  **Налаштуйте файл оточення:**
    Створіть `.env` у корені проєкту:

    ```env
    OPENAI_API_KEY=ваш_ключ_openai
    TELEGRAM_TOKEN=ваш_токен_бота
    ```

2.  **Встановіть необхідні бібліотеки:**

    ```bash
    pip install langchain-openai langgraph colorlog pydantic python-dotenv
    ```

3.  **Запустіть чат:**
    ```bash
    python src/main.py
    ```

## 🛠 Обробка помилок та логування

У проєкті реалізована багаторівнева система відстеження подій:

- **Зелений (INFO)**: Повідомлення про успішні операції та запуск.
- **Жовтий (WARNING)**: Попередження про порожні відповіді або дивну поведінку.
- **Червоний (ERROR)**: Детальні стеки помилок (tracebacks) у разі збоїв у логіці чи API.
